# Don't trust the executable? Build the setup yourself:

    ** Prequisite: Git, Node, Yarn package manager

- Clone the repo:

```bash
$ git clone https://github.com/ashishBharadwaj/flawesome.git
```

- Install the dependencies:

```bash
$ yarn install
```

- Build for production:

```bash
$ yarn build
```

- Create Package:

```bash
$ yarn package
```
